# Flappy Bird Clone (Unity)

A simple Flappy Bird–style game built using **Unity**.
You control a bird and try to fly between pipes without hitting them.
Built as a learning project for Unity and game development basics.

## 🕹️ How to Play

* Press **Space**.
* Avoid obstacles.

* Classic flappy mechanics
* Scoring system
* Random pipe generation
* Simple animations & sound effects
